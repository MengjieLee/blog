知识储备

这会是一个逐步深入，不断积累知识点和经验的过程

编程语言

  • Go: 高性能计算和分布式系统开发
  • Python: 数据处理和分析

    线性数据结构

  • Array: 连续存储，随机寻址高效（O(1)），插入删除低效（O(n)）
    • 分布式计算框架 Hadoop MapReduce、Spark 中用到 Array 存储批量数据，如日志切片、数值型特征等
    • 结合向量化运算，如 Spark 的 Vector 操作优化数据处理效率
  • Linked List: 非连续存储，随机寻址低效（O(n)），插入删除高效（O(1)）
    • 内存管理中模拟链表结构，如 Spark 的 内存缓存链，动态管理数据块
    • 分布式系统的元数据管理，如 HDFS 的 Block 分片，记录文件分块的位置信息

      树形数据结构

  • Binary Tree: 每个结点最多两个子结点，常见类型包括 Binary Search Tree、AVL Tree、Red-Black Tree 等
    • 分布式索引构建，如 Elasticsearch 的倒排索引 Inverted Index 基于红黑树优化
    • 数据排序与查找，在 Spark 的 sortByKey 中，底层可能用树结构优化排序性能
    • 数据库成本优化器(CBO) 使用决策树进行词法分析，生成最优执行计划
  • Heap: 完全二叉树，分为最大堆（父>=子）和最小堆（父<=子）
    • 海量数据 Top-N 问题，如用最小堆筛选日志中的高频词（MapReduce 流程 Map → Combine(Optional) → Shuffle → Reduce 中的 Combine 阶段的局部 Top-N），在 Map 端提前局部聚合并减少网络 IO 开销
    • 资源调度，YARN 的调度器基于最大堆从堆顶取出优先级最高的任务去分配资源
    • Spark Streaming 的事件时间窗口（Event Time Window）用堆管理超时数据

      集合与映射结构

  • Set: 元素唯一、无序（哈希集合）或有序（树集合）
    • 去重处理，如 Spark 的 distinct() operation 算子对海量用户 ID 去重（底层基于哈希集合）
    • 交集/并集计算，在用户分群场景中，用集合运算快速计算不同群体的重叠部分
